# Vision-Guided Robotic Sorting & Palletizing (Dobot MG400)

##Overview

This project implements a real-time vision-guided robotic manipulation system using a Dobot MG400 robot and Dobot Vision kit.

The system enables:
- Object detection via machine vision
- Real-time TCP/IP communication
- Autonomous robotic pick-and-place based on object classification

Two main workflows are implemented:
1. **FormNextVision** → Advanced object handling (Satellite, Engine, Dragon, etc.)
2. **Vision_Stacked** → Shape-based sorting and stacked object handling

---
Enables:
-Robot Control
-Dobot MG400 motion control via Lua scripting
-Cartesian motion using Jump() and MovL()
-Multi-step pick-and-place sequences
-Vision Integration
-TCP/IP communication with Vision Studio
-Real-time object detection input
